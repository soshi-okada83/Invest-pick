# 🧪 M0：UIモック実装

## 🎯 目的
- **「InvestPick」のトップページの見た目を固めることが目的**
- **まだバックエンドやニュースAPIには接続せず、ダミーデータを使って UI/UX を先に作り込むフェーズ**
- **Googleニュース（ダークテーマ）を参考に、「投資ニュース × 個人向けダッシュボード」っぽい雰囲気を再現する**

## 🧱 画面レイアウト

### ヘッダー

- 左：ロゴ  
  - 丸い青背景に **IP** の文字（ブランドロゴ）
  - 右隣にサービス名 **InvestPick** を大きく表示
- 中央：検索バー  
  - 「トピック、銘柄、ニュースを検索」というプレースホルダー
  - 現時点では検索機能は未実装（以降の拡張前提）

### コンテンツ

- 左：**サイドバー（ナビゲーション）**
  - 「ホーム / ウォッチリスト / セクター / AIトレンド / マクロ・政策」などのメニュー（メニュー名は変更する可能性アリ）
  - Googleニュース風の**丸みのあるボタン＋カードデザイン**
  - 下部に「投資は自己責任です」などの注意書き

- 右：**ニュースフィード**
  1. 上部に「今日のニュース」カード
     - タイトル：今日のニュース
     - 説明：ニュースを選ぶと詳細画面で「AI」に深掘り質問できる旨を記載
  2. 下にダミーニュース一覧（複数の `NewsCard` コンポーネント）
     - 各カードはソース名・時間・タイトル・概要・タグ（マクロ/AI/仮想通貨など）を表示
     - 全体が**角丸＋薄い枠線＋ダークグレー背景**で Googleニュース風のカードレイアウト
